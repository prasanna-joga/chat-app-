# Chat App - Vanilla JavaScript with Extended Functionality

A real-time chat application built using pure **Vanilla JavaScript**, HTML, and CSS, featuring extended functionalities commonly found in modern chat platforms â€” all without any external libraries or frameworks.

---

## Features

- Real-time messaging with instant updates  
- User authentication with unique usernames  
- Multiple chat rooms support  
- Message timestamps  
- Typing indicators showing when users are typing  
- Edit and delete messages  
- Emoji support  
- Auto-scroll to latest message  
- Responsive and mobile-friendly design  
- Persistent chat history using Local Storage  
- Message search functionality within chat rooms  
- Dark and light mode toggle  
- Notification alerts for new messages
